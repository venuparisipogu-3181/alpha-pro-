A real-time Options Screener using Dhan HQ API and Streamlit.

## Features
- Live PCR tracking for NIFTY, BANKNIFTY, FINNIFTY, and SENSEX.
- Automated Telegram Alerts based on PCR sentiment.
- Best strike selection based on Delta, Gamma, and OI.

## Setup
1. Clone this repo.
2. Install requirements: `pip install -r requirements.txt`
3. Create `config.py` with your credentials.
4. Run: `streamlit run app.py`
